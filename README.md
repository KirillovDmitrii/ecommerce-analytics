# E-commerce Analytics

Анализ данных интернет-магазина: исследование поведения пользователей, эффективности рекламных каналов и проверка гипотез.

## Содержание
- Предобработка данных (пропуски, дубликаты, выбросы, типизация, новые признаки)
- Исследовательский анализ (регионы, устройства, каналы, сезонность, типы оплаты)
- Расчёт ключевых метрик (средний чек, MAU, продолжительность сессии, топ-3 каналов и регионов)
- Проверка 7+ статистических гипотез (Kruskal-Wallis, Mann-Whitney, Spearman)
- Регрессионное моделирование продаж
- Дашборд в DataLens (5+ графиков, фильтры по каналу, дате, региону, типу оплаты)
- Презентация с ключевыми выводами и инсайтами (10 слайдов)

## Ключевые выводы
- Социальные сети — основной драйвер выручки и платящих пользователей
- Отсутствие повторных продаж (каждый клиент покупает 1 раз)
- Рекомендации по маркетинговой стратегии (блогеры, email, сезонность в Европе)
